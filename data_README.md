# Data

This directory should contain the WiDS Datathon 2026 dataset files:

- `train.csv` (221 fire–zone pairs, 37 variables)
- `test.csv` (95 observations, 35 variables, no outcome labels)
- `metaData.csv` (variable descriptions)

## How to obtain

1. Register at https://www.wids.io
2. Navigate to the WiDS Datathon 2026 wildfire prediction task
3. Download the dataset files and place them in this directory

**Note:** Raw data files are not distributed with this repository due to the competition's data usage terms.

Alternatively, set the `WIDS_DATA_DIR` environment variable to point to your data location.
